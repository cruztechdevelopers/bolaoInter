## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding |
| sm | 8px | Compact element spacing, gap between inline items |
| md | 16px | Default element spacing, card internal padding base |
| lg | 24px | Section padding, card body padding (p-6) |
| xl | 32px | Layout gaps between major sections (p-8) |
| 2xl | 48px | Hero section vertical padding on mobile (py-12) |
| 3xl | 64px | Hero section vertical padding on desktop (py-16), page-level spacing (space-y-20 = 80px) |

Exceptions: Touch targets on mobile buttons must be minimum 44px height (py-2.5 + text = ~44px). Modal overlay uses 0px margin on mobile (full-screen sheet), 24px margin on desktop.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Label | 12px (text-xs) | 400 (normal) | 1.4 |
| Body | 14px (text-sm) | 400 (normal) | 1.5 (leading-relaxed) |
| Heading | 20px (text-xl) | 700 (bold) | 1.2 (leading-tight) |
| Display | 36px (text-4xl) | 700 (bold) | 1.1 (leading-tight) |

Responsive scaling rules (do not introduce new base sizes):
- Display: 36px base, scales to 48px at sm breakpoint (sm:text-5xl)
- Heading: 20px base, scales to 24px at sm breakpoint (sm:text-2xl) where needed for section titles

Additional rules:
- Uppercase tracking-wider (letter-spacing 0.05em) for badges and status labels only
- Uppercase tracking-widest (letter-spacing 0.1em) for section eyebrow tags only
- Truncate (text-overflow: ellipsis) for user names and coupon codes in constrained layouts

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0a0a0a (--color-bg) | Page background, input backgrounds (#1e1e1e) |
| Secondary (30%) | #141414 (--color-bg-card) | Cards, sidebar, nav header, modal backgrounds |
| Accent (10%) | #10b981 (--color-primary) | See reserved list below |
| Destructive | #ef4444 (--color-danger) | Cancel checkout, error messages, logout confirmation |

Accent reserved for:
- Primary CTA buttons (solid bg-primary)
- Secondary CTA buttons (border-primary + bg-primary/10)
- Active tab indicators
- Points/score values in cupom cards
- Status badge "ativo" background tint (bg-primary/20)
- Ranking position highlight for user's own cupom
- Skeleton loading shimmer pulse color (primary/5 to primary/10)
- Toast success icon
- Live ranking pulse dot

Additional semantic colors:
- Warning: #f59e0b (--color-warning) — pending status badges only
- Gold: #fbbf24 (--color-gold) — 1st place ranking only
- Silver: #9ca3af (--color-silver) — 2nd place ranking only
- Bronze: #d97706 (--color-bronze) — 3rd place ranking only

Surface hierarchy (dark theme):
- Level 0: #0a0a0a (page bg)
- Level 1: #141414 (cards, modals)
- Level 2: #1a1a1a (card hover)
- Level 3: #1e1e1e (inputs, inner panels)
- Borders: #262626 (default), #333333 (light/hover)

## Layout Contract

### AppHeader (fixed, z-50)
- Height: 64px desktop, 56px mobile
- Background: #141414 (bg-card) with border-bottom border-border
- Logo: left-aligned, "Inter World Cup" text or trophy icon + text

- User menu: right-aligned, AvatarIniciais + name (desktop), AvatarIniciais only (mobile)

- Max width: 1024px (max-w-5xl), centered
- Header section: gradient bg from-primary-dim to-bg-card, rounded-2xl, p-6 sm:p-8

## Interaction Contract

### Page Transitions
- Fade transition: 200ms ease (opacity 0 to 1) between route changes
- Use Vue `<Transition>` with name="fade"

### Toast Notifications
- Position: fixed top-right, 16px from edges
- Enter: slide-in from right + fade, 300ms
- Exit: fade-out, 200ms
- Auto-dismiss: 4000ms
- Stack: max 3 visible, newest on top, 8px gap

### Skeleton Loading
- Show immediately on mount (no delay)
- Pulse animation: bg-bg-input to bg-border, 1.5s infinite
- Match exact dimensions of target content (cupom card shape, text line widths)
- Replace with real content via fade transition (200ms)

### Modal Auth
- Open: overlay fades in 200ms, modal slides up 300ms (mobile) or scales in 200ms (desktop)
- Close: reverse animations
- Close triggers: click overlay, press Escape, click X button
- Focus trap: tab cycles within modal when open

### Mobile Menu
- Open: overlay fades 200ms, menu slides from right 300ms
- Close: reverse
- Close triggers: click overlay, press Escape, click X, navigate to route

### Buttons
- Hover: bg-primary-hover (#059669) transition 150ms
- Active: scale(0.98) transition 100ms
- Disabled: opacity 50%, cursor not-allowed
- Loading: text changes to "Processando...", disabled state

### Cupom Cards
- Hover: border-border-light (#333) transition 150ms
- Status badge: ativo = bg-primary/20 text-primary, pendente = bg-warning/20 text-warning

### Responsive Breakpoints
- Mobile: < 640px (default, mobile-first)
- Tablet/Desktop: >= 640px (sm:)
- Large Desktop: >= 1024px (lg:)

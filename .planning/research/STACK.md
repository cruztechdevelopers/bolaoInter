# Stack Recommendation

## Backend

- Laravel
- PHP
- MySQL

## Frontend

- Vue 3
- Vite
- Vue Router
- Pinia

## Domain Conventions

- Código e domínio em português
- Tabelas em plural e `snake_case`
- Models em singular
- Campos estruturais do Laravel mantidos no padrão do framework

## Why This Stack

- Laravel reduz esforço em autenticação, migrations, validação, policies e jobs
- Vue 3 permite frontend responsivo com boa separação entre views, stores e composables
- MySQL atende bem ao domínio relacional de usuários, cupons, apostas, jogos e pontuação
